# Smart Library Management System

A console-based Library Management System developed in Java to automate library operations efficiently. The project implements Object-Oriented Programming principles, file handling, and user authentication to manage books and users. It allows administrators to manage library resources while providing users with secure access to available books.

## Key Features

* Admin Login
* User Registration & Login
* Add, View, Search, and Delete Books
* Issue and Return Books
* Persistent Data Storage using Text Files
* File Handling for Books and Users
* Menu-Driven Console Interface
* Object-Oriented Design

## Technologies Used

* Java
* OOP Concepts
* Collections Framework (ArrayList)
* File Handling
* Exception Handling
* Git & GitHub
